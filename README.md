# pruebaweb

A static web page built with HTML and CSS.

## How to view the page

### Option 1 – GitHub Pages (online)

The page is automatically deployed to GitHub Pages on every push to `main`.

1. Go to the repository on GitHub.
2. Click **Settings → Pages**.
3. Under **Source**, select **GitHub Actions**.
4. After the workflow runs, the page will be live at:
   `https://jsanchezv2201.github.io/pruebaweb/`

### Option 2 – Open locally (no server required)

1. Clone or download the repository.
2. Open `index.html` directly in your browser (double-click the file or drag it into the browser window).

### Option 3 – Local development server

If you have Python installed, run the following command in the project folder:

```bash
# Python 3
python -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
